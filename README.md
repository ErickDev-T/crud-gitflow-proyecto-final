CRUD GitFlow – Proyecto Final

Proyecto simple de CRUD en Node.js usando Express para ver el flujo de trabajo GitFlow, incluyendo ramas feature/, dev, qa y main, con Pull Requests y merges.

Requisitos
- Node.js instalado
- npm instalado

Instalar
npm install

Ejecutar el servidor
npm start

Acceder al proyecto
http://localhost:3000

/users → Lista de usuarios en JSON
